## 核心功能
### 配置管理
- API 配置：支持自定义 AI 服务的 API Key，可选择不同的 AI 模型
- 图片识别配置：独立开关及 API 配置，控制图片交互功能
- 对话参数设置：可调整最大上下文长度、总结触发次数、总结上限等参数
### 创建角色、群聊、世界
- 创建AI角色：可自定义角色名称、头像、背景、身份设定（角色身份、描述、行为规则等），已创建的角色也可以在聊天界面侧滑的抽屉中重新编辑~
- 创建群聊：可以选择多个已创建的的AI角色组成群聊，并为每个角色独立分配在群聊中的回复概率，角色在群聊中也会向单聊一样产生记忆，但可以放心的是所有记忆都是相对独立的！
- 创建世界：在创建群聊的时候可以选择一个已创建的世界（也可以后续添加，角色单聊也支持加入世界设定），可以将AI对话中的世界变成你所描述的样子；同时，你也可以添加一些世界专属的名词，不过记得写上相应的解释，当你在对话中提到这个名词时，将会被注入到发送的消息中方便AI理解

### 对话与交互
- 文本消息收发，支持用户与 AI 角色的纯文本对话（左：单聊  右：群聊）
- 点击[发送旁白]按钮，可以发送系统旁白，以此引导剧情往你希望的方向发展~可能会康到你想康的👀
- 图片交互：用户可选择本地图片发送给 AI，AI 可基于图片内容进行回复（需配置图片识别 API）

![c86f41f9b57a3303c00d2ce50aa9aaf4](https://github.com/user-attachments/assets/0c2affb1-2be6-4a1a-94d7-1957a698351c) ![2](https://github.com/user-attachments/assets/a5c57947-8bba-45d6-888e-4b8a6fc81c2c)

- 对话记忆与总结：当对话达到一定次数时，自动总结对话关键信息并保存为记忆片段，减少上下文长度同时保留重要信息
- 抽屉里面提供了丰富实用的功能
- 点击抽屉中对话设定的任意一项，都会来到会话管理中，这里可以为每个聊天单独设定“我”的个人形象（**是时候当一次帅哥了**），或者随时进行更换世界或调整群聊角色等操作
